Проект «Task manager» представляет собой разработку API с использованием Python и фреймворков Django. Основные функциональные требования включают управление постами и лайками. В рамках проекта необходимо выполнить следующие задачи:

  • Проектирование и реализация структуры базы данных для хранения постов и лайков.
  • Разработка API с указанными эндпоинтами:
      POST /posts — создание нового поста;
      PUT /posts/{id} — изменение существующего поста;
      DELETE /posts/{id} — удаление поста;
      GET /posts/{id} — получение информации о конкретном посте;
      GET /posts — получение списка всех постов;
      POST /posts/{id}/like — поставить лайк посту;
      DELETE /posts/{id}/like — убрать лайк с поста.
  • Реализация валидации входных данных.
  • Написание базовых unit-тестов для ключевых функций.
  • Настройка Docker для развёртывания проекта.
  • Подготовка README.md с инструкциями по установке и запуску проекта.
